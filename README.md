# Simple QRCode Generator

A simple, free and privacy-friendly web tool to generate QR codes directly in the browser.

## Features

- Generate QR codes from text or URLs
- Runs directly in the browser
- No account required
- No server-side storage
- Lightweight static web app
- Easy to deploy on Cloudflare Pages

## Privacy

This tool generates QR codes locally in the user's browser.

The text or URL entered by the user is not intentionally sent to external QR generation services, stored on a server, or logged by this project.
